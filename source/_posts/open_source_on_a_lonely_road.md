## 如何参与一个开源项目

很多人不知道如何参与一个开源项目，有的人以为只有贡献了源代码才算参与开源项目。这里我提供一个参与开源项目的步骤，以供参考：

1. 阅读项目的所有文档
2. clone 项目并在本地运行（build & test）
3. 使用过程中遇到问题可以提出 issue，**注意先搜索有没有相关的 issue**，不要提出重复的 issue
4. 遇到 bug 也可以提 issue，若有能力可以尝试解决并提出 pull request，**同样注意先搜索下看有没有别人已经提过了 pull request**
5. 若存在 [CONTRIBUTING](https://help.github.com/articles/setting-guidelines-for-repository-contributors/) 文件，则在提出 issue 或 pull request 之前需要先阅读它，里面会写贡献项目需要注意的事项
6. 关注（watch）项目的进展
7. 待对项目熟悉时候，就可以开始帮忙解决 issue，或者参与 issue 的讨论了
