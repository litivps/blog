# 流浪小猫的博客 / xcatliu's Blog

- Homepage: http://blog.xcatliu.com/
- Theme: [Milk](https://github.com/xcatliu/blog/tree/master/themes/milk)

## Getting Start

### Installation

```shell
npm install
```

### Run in localhost

```shell
npm start
```

Open your browser at http://0.0.0.0:4000

## Deploy

```shell
npm run deploy
```
