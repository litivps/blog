## Getting Start

### Installation

```shell
npm install -g hexo
```

### Serve in 0.0.0.0

```shell
hexo clean && hexo generate && hexo serve
```

Open your browser at http://0.0.0.0:4000

## Deploy

```shell
hexo clean && hexo generate && hexo deploy
```
